# Cover Routes

Use this file to decide which cover route to render. Keep `SKILL.md` thin; update route details here when the cover system evolves.

## Route Map

| 栏目 | route | 默认版本 | 标题结构 |
| --- | --- | --- | --- |
| 视频日记 | `video-diary` | `v1.3.1` | 主标题是当天主题，保留系列信息，但标题更短、描边更轻、强调信息层级。 |
| 碎碎念 | `suisuinian` | `v0.1` | 主标题固定为 `碎碎念`，副标题写当天随口聊的扩展说明；内容本身不限制话题数量。 |
| 读书笔记 | `reading-note` | `v0.1` | 主标题是书名 `《XXX》`，副标题写章节、观点或当日阅读感受。 |

## Title Modes

- `free-title`: `--title` 是主标题，适合 `video-diary`。
- `fixed-title`: 主标题来自 route 配置的 `fixedTitle`，`--title` 或 `--subtitle` 作为副标题，适合 `suisuinian`。
- `book-title`: `--book-title` 或 `--title` 自动格式化为 `《书名》`，`--subtitle` 作为副标题，适合 `reading-note`。

## Version Notes

### `video-diary v1.2`

- 建立视频日记的系列化封面方向。
- 顶部左侧是 `视频日记`，右侧是日期和 `持续记录 DayNN`。
- 中央主标题使用大号白字，黑色描边要细，不能像小广告。
- 黄色黑字标签用于补充说明。
- 保持当前封面形式，不因横封面/竖封面/片头占位而省略 `视频日记`、日期和持续记录信息。
- 保留强系列感，适合正式视频日记。

### `video-diary v1.3`

- 继承 v1.2 的系列结构：左上 `视频日记`，右上日期和 `持续记录 DayNN`，黄色标签保留。
- 主标题比 v1.2 更短、更轻，优先 2-6 个字，不把完整观点塞进标题。
- 标题描边减轻，背景压暗增强可读性，避免靠粗描边制造“小广告感”。
- 选帧优先找人脸偏右/偏下、左侧或上方有安静区的画面；不要直接用带字幕的成片帧。
- 当前 renderer 仍是中心标题模板，遇到正中大头画面会遮脸；后续如继续升级，应新增左侧/顶部安静区标题模板，而不是继续放大中心文字。

### `video-diary v1.3.1`

- v1.3 的封面字体 bugfix 版本。
- 修复复杂汉字在过粗字面和重复膨胀下内部空隙消失的问题，例如 `量` 这类笔画密集字。
- 主标题不强制单行或双行，按当天标题长度和画面效果选择；核心标准是清晰、圆润、可读。
- 标题优先使用更圆润的字体，取消额外字面膨胀，只保留轻描边和阴影。

### `suisuinian v0.1`

- 主标题固定为 `碎碎念`，不把每天的小标题放成主标题。
- 当天的小主题、情绪或一句话摘要放在副标题位置。
- 内容可以讲到哪里算哪里，不按视频日记的一条主线限制。
- 顶部左侧是 `碎碎念`，右侧只保留日期或 `随手记录`。
- 色彩比视频日记更松弛，使用浅绿色标签区分栏目。
- 后续可以根据实际作品反馈继续演进。

### `reading-note v0.1`

- 主标题优先是书名，格式固定为 `《XXX》`。
- 副标题用来扩展：章节、观点、问题、阅读感受或一句核心观察。
- 顶部左侧是 `读书笔记`，右侧显示日期或 `阅读记录`。
- 色彩更收敛，使用暖黄色标签。

## Update Rule

- 新增栏目路线：先在 `cover-routes.json` 新增 route，再在本文件补说明。
- 风格设计：先在 Pencil 完成 3:4 / 4:3 双画布和截图核验，再用
  `vp cover design` 登记 Pencil 源文件、预览图和 renderer tokens。
- 版本演进：保留旧版本配置，新增 `v0.2` / `v1.4.0`，不要直接覆盖已有版本。
- 渲染脚本参数应保持通用，不为某一天写死日期、标题或 Day。
