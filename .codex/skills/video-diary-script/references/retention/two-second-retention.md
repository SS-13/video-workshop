# Two-Second Retention Rule

## Definition

两秒跳出率 = 用户在看到视频 2 秒内滑走的次数 / 内容总播放次数。

Lower is better. It means the first screen, first sound, and first sentence gave the viewer enough reason to stay.

## Video Diary Interpretation

Do not turn the diary into a marketing script. The goal is to make the first two seconds clear and alive:

- The viewer should know what concrete thing is being discussed.
- The first sentence should contain a question, contrast, conflict, unusual observation, or high-value statement.
- The opening should not start with date, series background, long self-introduction, or too much explanation.
- The first screen should show the face/subject clearly and keep key text in the center-safe area.

## Opening Choices

Pick one opening type:

| Type | Use When | Pattern |
| --- | --- | --- |
| 需求导向 | useful workflow/tool/lesson | `我今天发现一个能解决 X 的办法。` |
| 共鸣导向 | emotion/life/work/learning | `我发现越是 X，反而越 Y。` |
| 稀缺高价值 | personal rare observation/story | `这件事我以前一直没想明白，今天突然通了。` |
| 热点结合 | tied to current trend/model/tool/event | `大家都在聊 X，但我今天感受到的是 Y。` |

## Script Gate

Before finalizing a teleprompter script, check:

- First sentence can be spoken within about 2 seconds.
- First sentence contains the actual hook, not a lead-in.
- Hook still sounds like the user's personal voice.
- No generic influencer phrases like `你一定要看到最后`.
- If the topic is abstract, open from a concrete life scene.

## Edit Gate

When editing:

- Avoid long silence before the first word.
- Avoid black screen, app watermark, or dead air at the start.
- Keep first subtitle short and center-safe.
- If a stronger first sentence exists later in the recording, consider cutting to it first when it does not distort meaning.

## Douyin Retro Gate

When data is available, review:

- two-second bounce rate when Douyin exposes it
- average watch time
- cover click rate
- first sentence / first frame / first subtitle

Record whether the opening was likely blocked by visual weakness, slow lead-in, or high topic comprehension cost.
