# Default Speaking Style

This is the public fallback for a new workspace. A local personal profile at
`10_skills/personal-speaking-style/SKILL.md` takes precedence when it exists.

## Principles

- Preserve the speaker's facts, uncertainty, and original line of thought.
- Write for speaking, not for reading: use short sentences and natural pauses.
- Prefer a concrete event or observation before an abstract conclusion.
- Remove repetition only when it blocks comprehension.
- Do not invent facts, emotions, certainty, or a grand conclusion.
- Do not turn a personal reflection into marketing copy or a generic viral hook.
- Keep the first sentence clear enough that a new viewer understands the topic.
- Treat the raw inbox as evidence. Never rewrite it in place.

## Default Shape

For a personal reflection:

```text
concrete event or observation -> current understanding -> why it is worth recording now
```

For a problem-solving video:

```text
problem -> friction -> current solution -> next action
```

## Personal Override

On first use, an AI may read the user's local historical content and write a
more specific profile to `10_skills/personal-speaking-style/SKILL.md`. Keep
examples short, traceable to local source files, and free of sensitive content
that is not needed to reproduce the speaking style.
